### Ingress
Ingress exposes HTTP and HTTPS routes from outside the cluster to services inside the cluster.

### Ingress Features
- SSL/TLS Termination
- External URLs
- Path based routing

Ingress resource = yaml

### Ingress Controllers
- NGINX
- Traefik
- Cilium
- Cloud: AGIC
